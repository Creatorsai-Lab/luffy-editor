Why creators choose Luffy Create

Built for people who explain things — engineers, educators, and content creators who need clear, animated visuals fast.

Most video editors are built for filmmakers, not for technical storytelling. Luffy Create flips that. It is a lightweight, open-source animated video editor designed around scenes, code, and diagrams — so you can turn a concept into a polished MP4 without wrestling with a heavyweight timeline tool or paying for a subscription.

  • Free and open source — MIT licensed, no subscription, no watermark, no locked features.
  • Offline-first — every project and export runs locally; nothing is uploaded to a server.
  • Made for technical content — syntax-highlighted code blocks, arrows, charts, and diagram-friendly shapes.
  • Frame-accurate animation — enter, loop, and exit effects with precise timing on a scene timeline.
  • Direct MP4 export — render 720p or 1080p video locally with FFmpeg, plus PNG and WebP stills.
  • Lightweight — fast to install, fast to launch, low on system resources.

Frequently asked questions

Everything you might want to know before downloading.

Is Luffy Create free? +
Yes. Luffy Create is completely free and open source under the MIT License. There is no subscription, no paywall, and no account required.
Which operating systems are supported? +
Version 1.0.0 ships a Windows installer. macOS and Linux builds are planned, and you can already run from source on any platform with Node.js installed.
Do I need an internet connection to use it? +
No. Luffy Create is offline-first. Editing and MP4 export run entirely on your machine — there is no cloud upload and no telemetry.
What video and image formats can I export? +
You can export MP4 video at 720p or 1080p, and still images in PNG or WebP. All rendering is handled locally with FFmpeg.
Is it good for code walkthroughs and tutorials? +
Yes. Luffy Create includes syntax-highlighted code blocks, typewriter and draw-path animations, arrows, shapes, and charts — purpose-built for technical and educational content.
Why does Windows show an "Unknown publisher" warning? +
The v1 installer is not yet code-signed. Click "More info" then "Run anyway". The full source code is public, so you can review or build it yourself.
Is my data collected or sent anywhere? +
No. Luffy Create collects no data and contains no telemetry. Your projects are stored locally on your own computer.
